    The Documentation

    Endpoints

    GET http://requestkittens.com/cats

    A randomized set of category-specific cat images. The meat of the API.

    Authorization required. Pass your API key as the 'Authorization' header.

    Parameter Accepted Values Default Notes
    emotion One of the 7 cat emotions. See below. null Leaving it blank will pull a random assortment of cat emotions.
    numOfResults Integer from 1 to 100 20 Results will always be an array, even if numOfResults is set to 1.
    imageSize thumb, small, medium, full small
    thumb A 200x200 square thumbnail.
    small Width set to 480px with preserved aspect ratio.
    medium Width set to 960px with preserved aspect ratio.
    full Original image size, max 3000px width.

    Sample Request

    http://requestkittens.com/cats?emotion=happy&numOfResults=2&imageSize=full

    Sample Response

    Notes: Because some kittens have a wealth of emotional data on their faces, emotions are an array of 1 or more value.

    GET http://requestkittens.com/cats/:id

    Fetch information about a specific cat. Returns all available image sizes.

    Authorization required. Pass your API key as the 'Authorization' header.

    Sample Request

    http://requestkittens.com/cats/553d327e594a1b5f1b13f198
